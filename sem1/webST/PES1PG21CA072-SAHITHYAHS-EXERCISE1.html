LISTS OF COURSES



COURSE - PROGRAMMING WITH PYTHON

        

    
        Unit 1: Computing Fundamentals - 11 Hours 
        Computational Problem Solving, Limits of Computational Problem Solving, Computer Algorithm, 
        Computer Hardware, Components of Digital Computer, Limits of IC Technology, Computer Software, Syntax, 
        Semantics and Program Translation.
 
        Unit 2: Procedural Python - 13 Hours
        Variables, Identifiers, Variable Types, Statements, Assignments, Expressions, Operators, Conditions, 
        Loops, Lists, Strings, Multi-dimensional lists, Tuples, Dictionaries, Sets, Strings, Output methods
 
        Unit 3: Python Functions - 10 Hours
        Functions, Recursive Functions, Exception Handling, Mapping, Filtering and Reduction, Lambda Functions, 
        List Comprehensions
 
        Unit 4: Python Object-Oriented  - 11 Hours
        Definition and defining a class, Constructor, Destructor, Classes and Objects, Inheritance, Polymorphism, 
        Error Handling and Exceptions - try, except and raise.
 
        Unit 5: File Processing - 11 Hours
        Reading and Writing Files, Deleting Files and folders, Reading csv, JSON files, Database Handling – PyMySQL and PyMongo



COUSRE - DISCRETE MATHEMATICS FOR COMPUTER APPLICATIONS


    
        Set Theory and Properties of Integers - (11 Hours)
        Sets and Subsets, Set Operations and the Laws of Set Theory, Counting and Venn Diagrams, 
        Principles of Inclusion and Exclusion, Primes and GCDs, Integers and Algorithms, Matrices, Mathematical Induction.

        Counting Principles  -  (11 Hours)
        The Rules of Sum and Product, Permutations, Combinations, Combinations with Repetition, The Pigeonhole Principle. 

        Fundamentals of Logic and Proofs -  (11 Hours)       
        Propositional Logic, Propositional Equivalences, Predicates and Quantifiers, Rules of Inference, 
        Introduction to Proofs, Normal Forms.

        Relations and Functions - (12 Hours)
        Relations: Cartesian Products and Relations, Relations and their Properties, Representing Relations, Closure of Relations, 
        Equivalence Relations, Partial Orderings.
        Functions: Plain and One-to-One, Onto Functions, Special functions, Function Composition and Inverse Functions.

        Graph Theory and its Applications  - (11Hours)
        Graphs and Graph Models, Some Special Types of Graphs, Representing Graphs, Connectivity, 
        Euler and Hamilton Paths. Introduction to Trees, Applications of Trees.



COUSRE - DATA STRUCTURES


       
        Introduction to Data Structures - 11 Hours                                                       
        Information and meaning –Abstract Data Types, Sequences as Value Definition, An ADT for varying length Character Strings, 
        Data Types in C, Pointers in C, Data Structures and C, Arrays in C, The Array as an ADT, Using 1-Dimensional Arrays,
        Implementing 1-Dimensional Arrays, Arrays as Parameters, Character strings in C, Two-Dimensional Arrays, Multi-Dimensional Arrays. 
        Structures in C, Implementing Structures, Representing other Data Structures
       
        Unit 2: Stacks - 12 Hours
        Definition and Examples
        Primitive Operations, The Stack as an ADT, Representing Stacks in C- Implementing the Push and Pop Operations, 
        An Example - Infix, Prefix, Postfix expressions, evaluating a Postfix Expression, Converting an expression from infix to postfix
       
        Recursion -
        Recursive Definition and processes - Multiplication of Natural numbers, The Fibonacci sequence, Binary Search, Recursion in C, Tower of Hanoi
       
        Unit 3: Queues - 10 Hours                                                                                                
        The Queue and its sequential representation – The Queue as an Abstract Data Type, C Implementation of Queues,
        The Priority Queue, Array Implementation of a Priority Queue, Circular Queue Using Dynamically Allocated Arrays
       
        Unit 4: Lists - 12 Hours
        Linked List- Inserting and Removing from a List, Linked implementation of stacks, The getnode and freenode operations, 
        Linked implementation of queues, The Linked List as Data Structure, Examples of List operations, Array Implementation of lists, 
        Linked List using Dynamic variables, Comparing the Dynamic and Array Implementation of Lists, Other List Structures, 
        Primitive operations on Circular List, The Stack as a circular list, The Queue as a circular list, Doubly Linked List.
       
        Unit 5: Trees, Sorting, Searching - 11 Hours
        Introduction, Terminologies, Representation of Trees, Binary Trees, Binary Tree Representations, 
        Operations of Binary Trees, Applications of Binary Trees, Binary Tree Representations, Implicit array Representation of Binary Trees, 
        Binary Tree Traversals in C, Inorder Traversal, Preorder Traversal, Postorder Traversal, Binary Search Trees Insertion into a BST, 
        Deletion from a BST, Shell Sort, Radix Sort, Hashing



COUSRE - PRINCIPLES OF OPERATING SYSTEMS


      
       Unit 1: Introduction to Operating Systems and its Structure   -  12 Hours
       What Operating System Do? - Computer System Organization, Types of Operating Systems, Operating System Operations, 
       Operating-System Services, System Calls, Types of System Calls, System Programs, Operating System Structure. 
       Linux Basics and System Startup: Installation, Basic Commands.
       
       Unit 2: Process Management and Process Scheduling - 11 Hours
       Process Management: Process Concept, Process Scheduling, Interprocess Communication, Process Scheduling - Basic Concepts,
       Scheduling Criteria, Scheduling Algorithms, Multiple Processor Scheduling. Linux: Process and Process Attributes, 
       Process metrics and process controls, listing processes, starting processes
       
       Unit 3: Process Coordination – Synchronization and Deadlocks - 11 Hours
       Synchronization - Background, The Critical-Section Problem, Petersons Solution, Synchronization Hardware, 
       Semaphores, Classic Problems of Synchronization;
       Deadlocks - System Model, Deadlock Characterization, Methods for Handling Deadlocks, Deadlock Prevention, 
       Deadlock Avoidance, Deadlock Detection, Recovery from Deadlock.
       Linux: Process Forking
       
       Unit 4: Memory Management - 11 Hours
       Memory-Management Strategies: Background, Swapping, Contiguous Memory Allocation, 
       Paging, Structure of the Page Table, Segmentation;
       Virtual-Memory Management: Background, Demand Paging, Page Replacement, 
       Allocation of Frames, Thrashing. Linux: Memory management commands
       
       Unit 5: Storage Management - 11 Hours
       File System - File Concept, Access Methods; Secondary-Storage Structure - Disk Structure, Disk Scheduling. 
       Linux-Files, File System architecture, Comparing files and File types, Access Permissions and ownerships.
   
        Text Book - Operating System Concepts​​​​​​​
        A. Silberschatz, P. Galvin , G. Gagne, 8th Edition, Wiley-India, 2009



COUSRE - WEB APPLICATION AND TECHNOLOGY


       

       Unit 1: Fundamentals of Web - 11 Hours ​​​​​​​
       Introduction to Web, The URL and Domain Name System, Overview of HTTP and FTP, The Hypertext Transfer Protocol, 
       The File Transfer Protocol, Cross Browser Compatibility Issues, W3C Validators, Website Design Issue, 
       Planning and Publishing of Website, Web Server. 
       Introducing HTML, Understanding Elements, HTML Document Structure Creating HTML Web Page, 
       Formatting Text with HTML Elements, Creating Hyperlink Using the Anchor Element, Inserting, Displaying List, 
       Using the TABLE, the DIV Element, the SPAN Element, Creating a Frame, Introducing Forms
       
       
       Unit 2: Designing webpages using Stylesheet - 11 Hours                                   
       Introducing CSS, Syntax of CSS, Inserting CSS in an HTML Document, Exploring Font Properties in CSS, 
       Applying CSS to Text, Applying CSS to Background, Exploring Link Properties in CSS, Positioning an HTML Element in CSS
       
       
       Unit 3: Programming with JavaScript - 12 Hours
       Introducing JavaScript, Using JavaScript in an HTML Document, Exploring Programming, 
       Fundamentals of JavaScript, Exploring Objects in JavaScript, Properties of an Object, 
       Methods of an Object, Exploring the Standard/Built-in JavaScript Objects, Exploring Browser Objects in JavaScript, 
       Document Object Model, Exploring Events, Validating an HTML Form Using JavaScript
       
       
       Unit 4: PHP- I ​- 11 Hours
       Features of PHP, Data Types in PHP, Determining Variable Data Type, Using Type Casting, Control Structures, 
       Conditional Statements, Looping Statements, Break, Continue, and Exit Statements, Built-In Functions in PHP, 
       String Manipulation Functions, Date and Time Functions in PHP, Mathematical Functions
       
       
       Unit 5: PHP - II - 11 Hours
       Building Web Applications Using PHP, Tracking Users Working with Forms and Database, Exploring Cookies, 
       Sessions and PHP Security, Introducing Databases, Exploring Relational Database Model, Records, and Primary Keys, 
       Understanding Logical Relationships of Tables and Concept of Foreign Keys, Using PHP and MySQL, Connecting to Database, 
       Selecting a Database, Adding Table in a Database, Altering a Table in a Database, Inserting Data in a Table, 
       Modifying Data in a Table, Retrieving Data.

   
       Text Book:
       “Web Technologies – BlackBook”, Kogent, Dreamtech Press
